## What is Jeremicah Licup?

Jeremicah Licup is a powerful documentation management platform that helps you create, organize, and maintain comprehensive project documentation. You can build structured docs sites with version control, search functionality, and collaboration tools. Whether you manage open-source projects, internal wikis, or customer-facing guides, Jeremicah Licup streamlines the process.

Use it to centralize all your Markdown files, MDX components, and custom pages into a professional, searchable knowledge base. Integrate with your Git workflow for seamless updates and previews.

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` with your email.
  </Step>
  <Step title="Connect Repository" icon="git-branch">
    Link your GitHub repo containing Markdown files.
  </Step>
  <Step title="Configure Site" icon="settings">
    Set your brand color (`#3B82F6`) and domain in the dashboard.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy your first docs site and share the live URL.
  </Step>
</Steps>

<CodeGroup tabs="Bash,Node.js">
  ```bash
  npm install -g @jeremicah-licup/cli
  jeremicah init my-docs
  ```
  ```javascript
  const Jeremicah = require('@jeremicah-licup/sdk');
  const docs = new Jeremicah({ apiKey: 'YOUR_API_KEY' });
  await docs.publish();
  ```
</CodeGroup>
